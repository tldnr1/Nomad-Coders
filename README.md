# 바닐라JS (JavaScript)
__유일한__ 프론트엔드 프로그래밍 언어  
- 할 수 있는 다양한 일
  - Library
    - 백엔드
    - 3D 구현 : three.js
    - 실시간 기능(채팅) : socket.io
    - 머신러닝 : ml5.js
    
  - Framework
    - React Native : 안드로이드, iOS 앱 개발 지원  (e.g. Discord, Instagram 등)
    - Electron : 데스크탑 앱 개발 지원  (e.g. VS Code, Twitch 등)
